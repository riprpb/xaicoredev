# XAICore Constitution: Security and Trust

Security is foundational and continuously improved. Enforcement belongs to the Kernel
and specialized services; intelligence may recommend but cannot silently create new
authority.

External files, messages, links, APIs, and other digital objects should pass through
the Trust Gateway where practical. Assessment focuses on necessary technical signals
and avoids unrelated surveillance or profiling.

Responses are proportional to risk: allow, warn, quarantine, request verification,
block, notify security systems, or escalate to the Owner.

KYC and financial security use data minimization, encryption, strict access control,
auditability, graduated trust, and applicable legal requirements. High-impact
financial decisions do not depend on one AI acting alone.
